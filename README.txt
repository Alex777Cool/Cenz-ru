# Cenz-ru
Проект сделанный для вычисления мата (на русском)
Для того, чтобы начать им пользоваться, вам необходимо скачать файлы, если у вас нет библиотеки для нечеткого сравнения fuzzywuzzy, нужно её установить (pip install fuzzywuzzy).
Затем мы заходим в файл "cenz-ru.py" и в переменную msg помещаем строку, в которой следует проверить на плохие слова (!!!ВАЖНО: значения, которые вы положите в msg должны быть типа str). 
Как бы не зотелось мне верить, но если очень долго посидеть над проектом, можно найти пару плохих слов, которые не получается распознавать. Для того, чтобы поместить дополнительные значения в код, нужно зайти в папку с проектом, удалить файл "cenz-ru.json", затем зайти в файл "cenz-ru.txt", добавить новые слова, или убрать старые и сохранить. После всех проделанных операций надо запустить файл "to_json.py". Появится файл "cenz-ru.json", в нем и будут все обновленные слова. Теперь можно спокойно запускать файл "cenz-ru.py" и все должно работать корректно
